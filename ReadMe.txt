Так как я счастливый обладатель лицензионного Windows 8 сначало пришлось Maven установить.

Установка Maven

Последнюю версию всегда можно скачать на странице загрузки на официальном сайте.
(http://maven.apache.org/download.cgi) 
Просто распаковываем архив в любую директорию. Далее необходимо создать переменную в Path, в 
которой необходимо указать путь к Maven. Заходим в Win + Pause – Дополнительно – 
Переменные среды – в верхнем окошке нажимаем Создать, вводим имя M2_HOME и значение допустим 
“C:\apache-maven-2.2.1”. Далее там же создаем еще одну переменную M2 со значением %M2_HOME%\bin. 
Так же убеждаемся, что есть переменная JAVA_HOME с путем к JDK. Ее значение должно 
быть примерно таким «c:\Program Files\Java\jdk1.6.0_10\». И наконец в том же окошке 
создаем/модифицируем переменную Path, в нее необходимо просто написать %M2%, чтобы наша папочка 
с исполняемым файлом Maven была видна из командной строки. Теперь необходимо проверить работоспособность нашей установки. 
Для этого заходим в командную строку и вводим команду 

   mvn -–version

Должна появиться информация о версиях Maven, jre и операционной системе, что-то вроде:

   Maven version: 2.2.1
   Java version: 1.6.0_10
   OS name: "windows 2003" version: "5.2" arch: "x86" Family: "windows"

Все, Maven готов к работе, можно приступать к созданию приложения.

Сборка проекта

Для этого заходим в командную строку и вводим команду 

	cd [путь до директории куда была выполнена команда «git clone http://studygit.simbirsoft1.com/serbinav86/homework.git»]
	
Произойдет смена каталога, затем вводим команду 
	
	mvn install
	
Должна появиться информация, что-то вроде:
	
	[INFO] ------------------------------------------------------------------------
	[INFO] Reactor Summary:
	[INFO]
	[INFO] Maven git application .............................. SUCCESS [  0.519 s]
	[INFO] printing_module .................................... SUCCESS [  2.020 s]
	[INFO] console_release .................................... SUCCESS [  0.871 s]
	[INFO] file_release ....................................... SUCCESS [  0.332 s]
	[INFO] console_app ........................................ SUCCESS [  2.773 s]
	[INFO] ------------------------------------------------------------------------
	[INFO] BUILD SUCCESS
	[INFO] ------------------------------------------------------------------------
	[INFO] Total time: 6.732 s
	[INFO] Finished at: 2016-02-15T22:16:40+03:00
	[INFO] Final Memory: 14M/166M
	[INFO] ------------------------------------------------------------------------
	
Затем вводим команду 

	cd [путь до директории куда была выполнена команда «git clone http://studygit.simbirsoft1.com/serbinav86/homework.git» + console_app\target]
	
Произойдет смена каталога, затем вводим команду 	

	java -jar console_app-0.0.1-jar-with-dependencies.jar
	
В консоли должно появиться сообщение
	
	Введите размера пиццы: 
	
Вводим число, если это будет не число будет выведена ошибка: Вы ввели не число, попробуйте повторить ввод еще раз! 
Если число будет отрицательным или равно 0 будет выведена ошибка: Вы ввели 0 или отрицательное число, попробуйте повторить ввод еще раз! 
	
	Вводите ингредиенты пиццы (чтобы завершить ввод введите вместо названия ингридиента введите - стоп )
	На складе имеется: сыр 100 колбаса 100 перец 100 
	Введите название ингредиента
	
Вводим название ингредиента, если ингредиента нет на складе будет выведена ошибка: Вы ввели ингредиент которого нет на сладе, повторите ввод еще раз
	
	Введите количество ингредиента
	Данного ингридиента на складе осталось 100
	
Вводим число, если это будет не число, будет выведена ошибка: Вы ввели не число, попробуйте повторить ввод еще раз! 
Если число будет больше, чем количество ингридиента на складе будет выведена ошибка: Вы ввели ингредиент  количества которого не хватает для приготовления пиццы, повторите ввод еще раз
Затем повторяется ввод названия ингредиента пиццы и его количество пока вместо названия ингридиента не будет слово стоп
После этого выводится сообщение(ниже приведен примерный текст сообщения)

	Вы заказали пиццу размером: 7 Состаящую из: сыр 50 перец 47 
	
Если на любом этапе работы программы ввести ? или help то появиться следующее сообщение
	
	Структура программы: Задать размер пиццы, задать название ингридиентов для пиццы и их количесво, распечатать состав и размер заказанной пиццы, 
	а теперь продолжайте ввод данных!
